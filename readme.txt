-: Description :-

Bookers is an Online Book Store. 
It has 2 roles 1) Admin, 2) User. 
User can login, signup, serch, add book into cart and purchase the itme in the cart.
Admin can perform all the users tasks and also he can add new admin or Normal user into the system.
Admin can also add books on the portal.

-------------------------------------------------------------------------------------------------


Requirements -

ASP.NET Core 5
SQL Server
npm
Visual Studio 2019


-------------------------------------------------------------------------------------------------

Installation


For Front-End :
cd FE
npm install
